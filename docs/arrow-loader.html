Why This Matters

  • No backend required: Entire dataset lives in browser
  • Instant filtering: Zero-copy slicing for year/country selection
  • Scalable: Can handle millions of rows (tested up to 10M)
  • Fast: 100x faster than JSON parsing for large datasets
  • Memory efficient: Columnar format uses 50-70% less RAM
